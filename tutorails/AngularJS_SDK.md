## Angular SDK 介绍

Loopback AngularJS SDK 包含三个重要组件:  
1. Auto-generated AngularJS services(自动构建 angualr的 services 服务), 依赖 ngResource
2. Command-line tools(命令行工具): `lb-ng`
3. grunt 插件(grunt-loopback-sdk-angular), 用以代替 `lb-ng`

构建工具将能与前端无缝结合:  

1. 注释: 拥有详细的 ngdoc 注释
2. 如果你高兴可以自己写构建方法
